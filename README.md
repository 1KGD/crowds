# Crowds (Working Name)

A colony sim game in the browser. Built with rust wasm-bindgen.

## Running Locally

> [!IMPORTANT]
> If you don't have them already, install cargo and make first.

```sh
# This will also install wasm-pack and build the rust "backend" before installing it.
npm i

# Builds production version and runs a preview server
npm run preview

#Alternativly, for a development enviorment
make install-dev
npm run dev # Run an automatically-refreshing development server
```

All assets are in the public domain.
